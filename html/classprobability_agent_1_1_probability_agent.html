Detailed Description

ProbabilityAgent class.

This agents implements a given Policy, which it uses to act. The agent takes into account probabilities for his actions.

The probability given is the probability that the agent will perfrom the action it desires. The other actions equally share the left- over probability.

E.g., if all actions are possible, and the agent has a probabilty of 0.7, the chances of the agent moving up, if it desires to do so, are 70%, and the chances for any of the other actions are 10%.
